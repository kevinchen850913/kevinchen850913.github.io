# VIVADO 啟動畫面

  • Quick Start (快速入門)
    Create Project → 創建項目
    Open Project → 打開文件項目 (.xpr)
    Open Example Project → 瀏覽範例程式
  • Tasks (任務)
    Manage IP → 新建或加載 ip
    Open Hardware Manager → 邏輯分析器 I/O 分析器
    Vivado Store
  • Learning Center (學習中心)
    Documentation and Tutorials → 文檔和教程
  • Recent Projects (近期項目)
    Recently visited projects → 最近訪問的項目
  • Tcl Console (Tcl 控制台)
    Tcl Console → TCL 殼

# VIVADO 專案畫面

  • Flow Navigator
    FPGA design flow
  • Sources
    RTL design /.tb
    constraints
  • Properties
  • Project Summary
