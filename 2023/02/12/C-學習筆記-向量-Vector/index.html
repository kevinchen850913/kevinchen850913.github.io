# Vector 介紹

Vector 是 C++ 標準程式庫中的一個類，vector 定義於 vector 標頭檔中。與其他 STL 元件一樣，vector 屬於 std 名稱空間。

# Vector 的設計目標

vector 是為了改善陣列的缺失與不便，而提供一種更有效、更安全的陣列。

# Vector 的特色

  • vector 的使用介面刻意模仿陣列，較明顯的差異在記憶體管理，陣列宣告時需明確指定長度，但是 vector 不需要，而是會在執行期依據狀況自我調整長度，動態增大容量。
  • vector 的表現一如資料結構中的陣列，允許隨機存取 (Random Access)，以索引值 (index) 存取任一元素只要花費常數時間 O (1)，在集合尾端增加或刪除元素也是花費常數時間 O (1)，若在 vector 集合中間增加或刪除元素時間複雜度是線性時間 O (n)，較為費時。

# 成員函式概觀

# vector 類別是以容器模式為基準設計的，容器可以三種方式看待。

  • 存取：即存取容器中物件的方式。
  • 儲存：即儲存容器中物件的方式。
  • 遍歷：即遍歷容器中物件的方式。

# 典型的容器實現的方法如下

  • 建立一個新的空容器 (建構函式)
  • 向容器中插入物件
  • 從容器中刪除物件
  • 刪除容器中的所有物件 (清空)
  • 存取容器中的物件
  • 取得容器中物件的數目 (尺寸)

# vector 類別的容器實現的方法如下

  • 存取元素的方法

以 at () 存取會做陣列邊界檢查，如果存取越界將會拋出一個例外，這是與 operator 的唯一差異。撰寫較嚴肅、安全性較高的程式時使用 at ()。

  • 新增或移除元素的方法
  • 取得長度 / 容量
  • 重新組態／重設長度

容量與記憶體的組態有關，不會因為元素被刪減而隨之減少，
在眾多的 STL 實例，容量只能增加，不可以減少。
