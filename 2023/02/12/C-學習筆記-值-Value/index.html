# 字面值 (Literal Value)

字面值是代表固定值的符號，通常用來初始化變量，例如： int i = 1 ，1 是整數字面值。

# 種類

# 特性

  • 重要的
  • 優先配置的
  • 內容不變
  • 位置不變

# 只要是值就一定有型別

# 常量 (Constant)

用符號表示數據，一旦確定了表示哪個數據，以後就不能再改變。

# 探討 volatile 和 const 可否同時存在？

可以的，假設今天是一個唯讀狀態暫存器，宣告為 volatile 因為他可能被意想不到的修改掉，宣告為 const 因為我們的程式不能修改他的值 (唯讀)。

# 變數 (Variable)

# 宣告

要求配置記憶體空間 註冊新名稱。

# 指派

將右值指派給左值。

# 取值

只要變數名稱不是出現在 = 左側，便是取得變數所代表的記憶體中的值。

# 隱式類型轉換 (Implicit conversions)

隱式類型轉換也稱為自動類型轉換，編譯器自行完成，無需用戶的任何外部觸發。

  • 通常在表達式中存在多種類型時發生，會類型轉換以避免丟失數據
  • 數據類型會升級為更大的數據類型
  • 隱式轉換可能會丟失信息，並且可能發生溢出

# 顯式類型轉換 (Casting)

顯式類型轉換也稱為類型轉換，用戶可以進行類型轉換，使其成為特定的數據類型。

  • 這樣做可利於應用結構或類型的某些特性
  • 助於計算包含不同數據類型變量的表達式
