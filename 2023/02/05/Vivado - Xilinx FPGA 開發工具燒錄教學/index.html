# 燒錄教學介紹

本篇文章說明燒錄程式的方法，從專案產生 bin 檔開始，接著把 mcs 檔燒進非揮發性記憶體，燒錄成成功後重新上電，及可成功載入程式碼。

連接裝置方法一，直接從快速目錄進入 Hardware Manager

Open Target & Auto Connect

連接裝置方法二，從專案中進入 Hardware Manager

# 燒錄 bin 檔

可以先連線燒錄 bin 檔，測試完功能正常後再產生 mcs 檔。

# 產生 mcs 檔

使用工具將 bin 檔傳換成 mcs 檔，專案編完時也有跳窗選項。

選擇要燒錄的記憶體型號，Vivado 燒錄的問題之處，可以先記住。

# 燒錄 mcs 檔

Adding Configuration Memory Device，如果之前就已經燒錄過檔案，可以先將之前的檔案清除。

Selecting Configuration Memory Device，選擇要燒錄的記憶體型號，怎麼感覺似曾相似，還記得前面就有提過這個問題，這件事就是要做兩次沒錯，而且這串問題還有後續。

Programming Configuration Memory Device，這系列問題的最嘲諷之處，如果記憶體型號選錯，會跳出錯誤跟你說正確的型號，看到這句話就是代表你要從產生 mcs 開始重做，不是阿你都可以讀到正確的型號了，還要我自己輸入，而且還要輸入兩次，你到底有什麼毛病？
