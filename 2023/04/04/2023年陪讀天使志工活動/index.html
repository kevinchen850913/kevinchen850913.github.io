# 莊家姊弟介紹

從國外到台灣的莊家姊弟，不到一年就變成真正的台灣人，中文能順利溝通，課業也慢慢地追上，跟奶奶一起看狗血鄉土電視劇，說幾句台語也沒問題，已經順利的融入這裡的生活，不過聽說小學畢業後就又要搬出國，有夠國際化的學習經歷。

弟弟作勞作時有個很奇怪的習慣，他不喜歡自己動手做，喜歡找別人幫忙，志工 A 你能不能幫我？志工 B 你能不能幫我？然後他的作品就完成了，而他在作品中扮演的角色是監工，你要折好一點喔，你要剪漂亮一點喔，到下課時間媽媽來接他，此時他興奮的拿著作品對媽媽說，你看這是我做的！

姊姊在完成作業上對自己很有要求，常常很晚才能完成作業，日記內容不好重寫，字不漂亮重寫，不過感覺是過程大於結果，有時候考試還是會考砸，不過學生自己能接受就好，我小時候就會顧好成績，快速完成作業，然後就跑去玩了，有獎勵會比較有動力處理課業。

她教我要怎麼做一朵花，她炫耀自己的作品時看起來很有成就感，但其實她是不太喜歡拍照的學生，她有很多自己的小心思，今天頭髮會不會太亂，假日是不是又曬黑了，哪裡又不滿意了，雖然我都看不出差異，不過這麼說她會有點生氣，反正你都不懂啦。

# 蘇家偶爾才來

蘇家自從疫情結束後就很來活動，每天看的小孩長大都沒感覺，一年沒看的小孩，感覺一下就長大了，雖然還是喜歡往箱子鑽，不過已經很難再變魔術，箱子已經無法完全闔上，還好我之前有照片存證，有好好地留下成長的點點滴滴，不知道下次見面又待何時。

今天就只有蘇家小弟來，三個志工加一個學生就開始玩起了疊杯子的遊戲，一開始我還搞不懂這種遊戲到底有什麼好玩，結果玩著玩著就投入其中，果然遊戲好玩是因為參與的人，而不是遊戲本身。

# 美術班與網癮少女

去年自從考上美術班就很少參加活動，今年好像都沒有看到她，因為從小就是鑰使兒童，所以一直都是手機不離身，隨著年齡的增長，對手機的依賴性也隨之提升，不知道她最近過得如何，應該要來關心一下近況。

好閨密互相影響，考美術班與沉溺於科技產品，網路社交大師，各種遊戲樣樣來，像我幾乎不在網路上進行社交，因為珍惜身邊的人時間就不夠了，哪有餘力去維持萍水相逢的人際，但還好她並不是那種比較冷酷無情的人，不然我又會倍受打擊，交朋友還是要找重情重義的比較好

我有 w、傳、極、蛋仔，很高興認識你喔！歡迎來加遊戲好友，喜歡花子君，有閨、缺男友。這是她在網路上的自我介紹，看來我是不是也該找幾個遊戲來玩，感覺我跟現在的小朋友已經完全脫節。

# 第四年的反思

今年是餐與陪讀天使活動的第四年，一路走來收穫了很多快樂，每年都在經歷相逢與別離，我有預感這段緣份可能要到盡頭了，我自己工作上的變動，陪讀活動招生的不作為，今年學生屈指可數，若學生不參加，這個活動只能草草收尾，最讓我放不下心的就是這些學生，可能等到塵埃落定後再來補上結局。
