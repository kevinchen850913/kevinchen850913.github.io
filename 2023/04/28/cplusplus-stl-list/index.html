# List 介紹

list 是 C++ 標準程式庫中的一個類，也被稱為雙向鏈表（doubly linked list），list 定義於 list 標頭檔中，它可以儲存任意型別的元素，並支援在任意位置插入、刪除、查詢元素等操作。

list 中的每個元素都包含了指向前一個元素和後一個元素的指標，這使得 list 在插入、刪除元素時效率高，因為不需要移動其他元素。

C++ 的 List 不支援 List[] 用法，要特別注意！
所以不能像陣列那樣可以直接用索引值存取。

# 個人心得

List 不支援 List[] 用法有點麻煩，一般情況直接無腦用 Vector，所以 C++ 的 Vector 用法比較接近 C# 的 List 。
List : 插入刪除比較快，讀寫值比較慢。
Vector : 插入刪除比較慢，讀寫值比較快。
