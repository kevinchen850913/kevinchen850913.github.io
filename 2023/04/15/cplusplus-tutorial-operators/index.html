C++ 學習筆記 - 運算符(Operators)

發表於2023-04-15|更新於2023-10-29
|閱讀量:

# 賦值運算符

此語句將整數值分配 y 給變量 x ，總是從右到左進行。

# 算數運算符號

  • 檢測左值是否為右值的倍數
  • 非 10 進位之週期運算
  • 取得連續範圍的整數
  • 位置參照

# 遞增與遞減運算符號

  • 單一敍述變數值增量。
  • 複合敍述符號在前面增量先運算，符號在後面增量後運算。

題目 : i = i+++i;

  1. i++ 會優先處理，但它屬於後加型。
  2. 理計算和的動作 i + i 。
  3. 後加型 i++ 再將 i 加 1。

# 複合指派運算符號

運算後指派，將左值與右值運算後的結果指派給左值

# 關係運算符號

運算的對象是數值，運算的結果是布林值 (bool)。

# 條件運算符號

運算的對象為布林值，運算的結果也是布林值

  1. && (And) : 左值不成立，右值不運算
  2. || (Or) : 左值成立，右值不運算
  3. ! (Not) : 用在變數或函式呼叫前面

# 三元條件運算符號

條件後面跟著問號，如果是在冒號前的表達式會被執行，如果否在冒號後面的表達式會被執行，這個運算子常常被用來當作 if 的簡潔寫法。
