# Vector 介紹

vector 是 C++ 標準程式庫中的一個類，vector 定義於 vector 標頭檔中，與其他 STL 元件一樣，vector 屬於 std 名稱空間。

vector 是為了改善陣列的缺失與不便，而提供一種更有效、更安全的陣列，較明顯的差異在記憶體管理，陣列宣告時需明確指定長度，但是 vector 不需要可動態增大容量。
