C++ 學習筆記 - 集合(Set)、無序集合(Unordered Set)

發表於2023-05-12|更新於2024-11-14

# Set 介紹

在 C++ 中，Set 是一個標準模板庫 (STL) 容器，它是用來存儲具有固定升序的元素集合，不允許重複的元素存在。Set 提供了許多操作，如插入元素、查找元素、刪除元素、獲取集合大小等等。此外 Set 還支持交集、聯集和差集等集合運算。

unordered_set 是一個與 set 用法類似的標準模板庫容器，提供了一個無序的集合，不允許重複的元素存在。

  • 如果要使用自定義類型，需要自己定義哈希函數。
  • unordered_set 是無序的在遍歷時不能保證元素的順序。

# unordered_set 跟 set 的差異

  • unordered_set 的實作方式為雜湊表 (hash table)，Set 實作方式為紅黑樹 (Red-Black Tree)
  • unordered_set 無序，set 有序
  • unordered_set 插入和查詢 O (1)，set 插入和查詢 O (log n)
  • unordered_set 消耗較多的記憶體

# Set 插入元素

set、unordered_set 容器中沒有 at ()、operator [] 功能，位置與數值相關，所以無法指定為值讀寫數值，不如說因為資料是進了紅黑樹或雜湊表，所以不會有固定已知的位置，自然無法隨機讀取某元素，總是會需要遍歷才能取出。

# Set 刪除元素

set、unordered_set 可以刪除不存在的元素，刪除了沒有，C++ 11 會回傳擦除的元素數。

# Set 的遍歷

set 容器裡面會幫你排序，輸出結果是由小到大排列，set 容器裡面會幫你排序，在插入元素的同時會根據元素來進行排序，而 unordered_set 遍歷時不能保證元素的順序。

# Set 判斷元素是否存在

  • count : 存在該元素的話回傳 1，不存在的話回傳 0。
  • find : 回傳指向該元素的 iterator
