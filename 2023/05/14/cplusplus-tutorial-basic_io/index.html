# 基本介紹

在 C++ 中，要從控制台獲取輸入和輸出，可以使用標準輸入和輸出流對象。標準輸入流對象 cin 和標準輸出流對象 cout 分別用於從鍵盤獲取輸入和向屏幕輸出信息。

# 流 (stream)

C++ 使用一種稱為流 (stream) 的抽象詞來描述媒體，例如螢幕、鍵盤或文件，流是一個實體，可執行輸入或輸出操作，無需了解流的詳細運作方式，只需要知道流的來源與目的地即可進行應用。

# 標準庫中的流

cerr 並且 clog 也是輸出流，區別是識別特定目的，它們可以單獨重定向，在大多數環境設置中，它們實際上做的是與 cout 完全相同的事情。

# 標準輸出 (cout)

在大多數環境中標準輸出是螢幕，用於訪問的是 cout

多個插入操作 << 可以鏈接在一個語句中

要插入換行符，換行符應插入到換行符的確切位置。
在 C++ 中換行符可以指定為 \n ，或是使用 endl

# cin 和字串 (string)

可用 cin 與基本數據類型相同的方式獲取字符串 (string)。

cin 提取會將空格、換行符視為提取的終止，因此終提取單個單詞，而不是短語或整個句子，若要獲取整行可用函數 getline 。

# 字串流 (stringstream)

標準標頭 <sstream> 定義了一個名為的類型 stringstream ，該類型允許將字符串視為流。
