# 循環語句

# while

如果 while 的 () 中為真，while 重複執行 {} 中的內容，若執行任何後不再為真，則循環結束。

該段程式功能為倒數，從 10 開始輸出，直到 n > 0 的條件不成立就結束。

while 要考慮是如何結束，因此應以某種方式更改條件中檢查的值，否則循環將永遠執行下去。

# do while

do while 與 while 的差異

  • 在執行之後評估，而不是之前評估。
  • 保證至少執行一次，即使永遠不會為真。

建議統一用 while，do while 比較難閱讀。

# for

循環重複 while 為真 + 計數器變量。

若要使用計數器，for 會比 while 更好閱讀，for 要特別注意空格用法，最好統一寫法。

# for each

for 循環還有另一種語法，專門用於遍歷容器中的所有元素。

通常元素類型會使用類型 auto，最好不要遍歷中修改容器中的元素。

# 跳轉語句

# break

break 會離開一個循環，它可用於結束無限循環。

# continue

continue 會跳過循環的其餘部分，類似直達語句塊末尾。

# goto

goto 允許絕對跳轉到程序中的另一點，是比較底層的功能。

不要使用 goto，無條件跳轉容易產生問題。

# 選擇語句 : if 和 else

# if

關鍵字 if 當且僅當滿足條件時，執行語句或塊。

# else

else 如果上述條件不為真則執行。

# 尤達條件式

尤達條件式（也稱為尤達標記法）是一種計算機編程中的編程風格，在此風格中表達式的兩個部份與條件語句中的順序將會對調，並且表達式的常數部份將會放在條件語句的左側。這種風格的名稱來自於星際大戰的絕地大師尤達，他使用著缺乏標準語法的英語。

# 選擇語句 : switch

# switch

switch 的目的是在許多可能的常量表達式中檢查一個值，類似於連接多個 if else 語句。

# 等效的 if else

範例為與 switch 語句等效的 if-else。

# 標籤特性

switch 從早期的 C 編譯器中繼承特殊的語法，因為它使用標籤而不是塊。這意味著語句之後需要 break 去離開塊。
