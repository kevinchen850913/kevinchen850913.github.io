C++ 學習筆記 - 映射(Map)、無序映射(Unordered Map)

發表於2023-05-13|更新於2024-11-14

# Map 介紹

在 C++ 中 map 是一個標準的容器，它提供了一種關聯式容器 (鍵和值)，通常情況下鍵可以是任何可比較的類型，而值可以是任何類型，由於 map 使用紅黑樹實現，所以在插入和查找操作方面具有較好的性能。

Map 具有以下特性

  • 基於紅黑樹因此查找、插入、刪除操作的均為 O (log n)。
  • 所有的 key 都是唯一的且順序排列。
  • 可以通過 key 快速查找對應的 value。
  • map 支持隨機存取、迭代器遍歷等操作。

# Unordered Map

在 C++ 中，unordered_map 是一個標準模板庫容器，提供了一個無序的映射，它提供了一種關聯式容器 (鍵和值)，通常情況下鍵可以是任何可比較的類型，而值可以是任何類型。

Unordered Map 跟 Map 的差異

  • unordered_map 的實作方式為雜湊表 (hash table)，map 實作方式為紅黑樹 (Red-Black Tree)
  • unordered_map 無序，map 有序
  • unordered_map 插入和查詢 O (1)，map 插入和查詢 O (log n)
  • unordered_set 消耗較多的記憶體

# Map 初始化
