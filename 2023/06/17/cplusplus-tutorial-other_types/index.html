C++ 學習筆記 - 其他數據類型

發表於2023-06-17|更新於2023-10-29
|閱讀量:

# 類型別名 (typedef /using)

C 語言繼承而來的關鍵字 typedef

C++ 新增的關鍵字 using

typedef 和 using 在語義上是等價的，唯一的區別是在沒有 typedef 模板的領域中有某些限制。常見的用法是抽像底層類型，通過使用別名來引用特定類型的參數，以後可較輕鬆替換類型。

# 聯合 (Unions)

聯合允許將內存的一部分作為不同的數據類型進行訪問。它的聲明和使用類似於結構。

這些成員中的每一個都是不同的數據類型。但由於它們都指向內存中的同一位置，因此修改其中一個成員將影響所有成員的值。不可能以每個獨立於其他值的方式在它們中存儲不同的值。

# 枚舉 (enum)

枚舉類型是使用一組自定義標識符作為可能值定義的類型。這些枚舉類型的對象可以採用這些枚舉數中的任何一個作為值。

# 枚舉類 (enum class)

在 C++ 中可以創建既不能隱式轉換為 int，也不能具有 type 的枚舉值，而是 enum 類型本身的實類型，從而保持類型安全。
